# HR Dashboard Analysis

## Overview
This dashboard analyzes 2,845 employee records from the HR dataset.

## Key Findings
- Total Employees: 2,845
- Production department has the highest number of employees.
- Employee distribution is spread across multiple business units.
- Most employees are currently active, while a smaller percentage are terminated.

## Visualizations
- Department-wise Employee Count (Bar Chart)
- Business Unit-wise Employee Count (Column Chart)
- Employee Status Distribution (Pie Chart)

## Conclusion
The dashboard provides insights into workforce distribution, department strength, and employee status.
